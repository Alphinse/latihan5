Kursus Komputer

Dalam kursus ini, Anda akan mempelajari dasar-dasar penggunaan komputer, termasuk sistem operasi, manajemen file, dan perangkat lunak produktivitas seperti Microsoft Office. Kursus ini cocok untuk pemula yang ingin memperdalam pemahaman mereka tentang komputer dan meningkatkan keterampilan mereka dalam menggunakan perangkat keras dan perangkat lunak komputer.

  • Pengenalan Sistem Operasi (Windows, macOS)
  • Manajemen File dan Folder
  • Penggunaan Microsoft Office (Word, Excel, PowerPoint)
  • Pengaturan dan Pemeliharaan Komputer

Kursus Web Programming

Kursus ini dirancang untuk mengajarkan Anda bagaimana membangun website yang interaktif dan responsif menggunakan HTML, CSS, dan JavaScript. Anda juga akan diperkenalkan dengan framework web seperti Bootstrap dan Tailwind CSS, serta belajar cara mengelola data dengan menggunakan API.

  • Dasar-dasar HTML5 dan CSS3
  • Pembuatan Layout Responsif dengan Flexbox dan Grid
  • JavaScript untuk Interaktivitas
  • Pengenalan ke Framework (Bootstrap, Tailwind CSS)
  • Menggunakan API untuk Pengelolaan Data

Kursus Android Studio

Kursus ini akan membimbing Anda melalui pengembangan aplikasi Android dari awal hingga akhir menggunakan Android Studio. Anda akan mempelajari cara membuat aplikasi yang intuitif dan menarik dengan berbagai komponen Android, serta mengoptimalkan aplikasi Anda untuk berbagai jenis perangkat.

  • Pengenalan Android Studio dan Struktur Proyek
  • Pembuatan Layout dan Desain UI
  • Pengelolaan Aktivitas dan Fragment
  • Implementasi Database dengan SQLite
  • Publikasi Aplikasi ke Google Play Store
